The community kept its product data scattered across 75 spreadsheets — impossible to search, full of duplicates. kina-search pulls it all together into a searchable database with over 100,000 deduplicated entries.

Crawlers run automatically via GitHub Actions and keep the data current without manual intervention. The frontend is a Next.js app with fast search across the whole dataset.
