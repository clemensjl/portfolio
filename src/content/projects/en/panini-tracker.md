Anyone collecting the Panini album for the 2026 World Cup quickly loses track: which stickers are missing, which are duplicates, what can be traded? panini-tracker answers that in a few clicks.

Deliberately minimal: a single HTML file, no build step, hosted on GitHub Pages. Collection progress stays local in the browser.
